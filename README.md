# TextNuvem ☁️

**TextNuvem** é um editor de texto e editor de código online em desenvolvimento, projetado para permitir a criação e gerenciamento de múltiplos projetos em um único ambiente.

A proposta é oferecer uma experiência leve, moderna e produtiva para escrita e organização de código diretamente no navegador.

## 🚧 Status do Projeto

> Projeto em fase ativa de desenvolvimento.

Novas funcionalidades estão sendo implementadas continuamente e a estrutura da aplicação ainda está evoluindo.

## ✨ Funcionalidades (atuais e planejadas)

* Editor de texto online
* Editor de código no navegador
* Gerenciamento de múltiplos projetos
* Persistência de arquivos e projetos
* Interface web interativa
* Recursos voltados para produtividade *(em desenvolvimento)*

## 🛠 Tecnologias Utilizadas

### Backend

* ASP.NET Core
* C#

### Frontend

* Blazor WebAssembly
* Razor Components

## 🎯 Objetivo do Projeto

O objetivo do TextNuvem é construir uma plataforma que combine:

* Escrita e edição de código online
* Organização de projetos em nuvem
* Uma experiência semelhante a um mini ambiente de desenvolvimento web

## 📌 Roadmap (Planejado)

* [ ] Sintaxe destacada (syntax highlighting)
* [ ] Auto-save
* [ ] Execução de código (estudo futuro)
* [ ] Compartilhamento de projetos
* [ ] PWA / suporte offline

## 🚀 Executando o projeto

```bash
# Clonar o repositório
git clone https://github.com/seu-usuario/textnuvem.git

# Entrar no diretório
cd textnuvem/frotend/

# Executar backend
dotnet run
```
